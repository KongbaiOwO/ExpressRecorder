# ExpressRecorder
快递打包视频监控软件

由Deepseek、通义千问大模型开发而来，分享给大家，如果大家有需要可以下载或者拿代码让ds继续完善（反正这样我够用了就行，至少是免费的，外面一年99或者一个月20（

需要Nuget包：
  - AForge.Video
  - AForge.Video.DirectShow
  - ZXing.Net
需要：ffmpeg（要自己搭好环境哦）

用法：
  扫描到快递单号，弹窗出快递单号和快递公司，如果不对可以进行修改，对的话点击开始录制即可录制打包封箱视频。点击结束即可结束。很简单的一个软件√
